🚨 Problem Statement

Modern households face growing safety challenges:

Increasing number of nuclear families

Working parents away for long hours

Elderly living independently

Lack of affordable real-time safety monitoring systems

High-end surveillance systems are expensive, complex, and not proactive.

💡 Our Solution

AI-Powered Smart Guardian is an autonomous IoT-based mobile monitoring robot designed for real-time home safety.

It combines computer vision, embedded systems, and alert automation to provide intelligent protection.

🔍 Key Features

✅ Real-time fall detection for children & elderly
✅ Instant evidence capture (image snapshot)
✅ SMS alert notification to guardians
✅ Dangerous animal detection (e.g., stray dogs, snakes)
✅ Live video streaming via ESP32-CAM
✅ Mobile, portable robotic platform

🛠️ Tech Stack

Hardware

ESP32 (Motor control & processing)

ESP32-CAM (Live streaming module)

Motor Driver + Chassis

Li-ion Battery Pack

Software

YOLO Object Detection Model (optimized for embedded use)

Python (Model training & inference)

Embedded C / Arduino Framework

SMS Alert Integration API

⚙️ System Architecture

Camera captures live video feed

YOLO model performs real-time object & fall detection

If fall/danger detected:

Image snapshot captured

SMS alert sent to guardian

Live stream accessible remotely
